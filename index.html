Decision Guide

When should you use Context7?

Think of it like choosing between asking a local for directions vs. using Google Maps. Sometimes memory is fine, sometimes you need live data.

‚úì Use Context7 when...

  • Working with new or recently updated tools
  • The AI's answer doesn't seem to work
  • You need the "current" way of doing something
  • Asking about tools released in the last 1-2 years
  • You keep getting "that doesn't exist" errors
  • The software version matters (v14 vs v15, etc.)

‚Äî Skip Context7 when...

  • Asking general knowledge questions
  • Working with old, stable tools that rarely change
  • The AI's answers are already working fine
  • You're brainstorming or exploring ideas
  • Asking about concepts, not specific tools
  • Quick prototyping where "close enough" is okay
üí°

Rule of thumb

If you're asking "how do I use [specific tool/software]?" and the AI's answer doesn't work ‚Äî try adding "use context7" and ask again. It's like asking someone to double-check the manual before answering.
